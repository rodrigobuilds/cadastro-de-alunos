**Sobre o projeto**

Este projeto consiste em um sistema desktop para gerenciamento de alunos, cursos e turmas, desenvolvido em Python com interface gráfica utilizando Tkinter.

A aplicação permite realizar operações completas de CRUD (Create, Read, Update e Delete), incluindo cadastro, edição, exclusão e consulta de registros, além de funcionalidades como busca de alunos e upload de imagens.

Os dados são persistidos em banco SQLite, garantindo simplicidade e eficiência no armazenamento.

O projeto foi desenvolvido com foco em simular um cenário real de sistema acadêmico, aplicando conceitos fundamentais de desenvolvimento de software, organização de código e integração entre interface e banco de dados.
